### Key Points:
1. **HP NIC Proxy** - Handles public SSL with a trusted government certificate
2. **PROD Nginx** - Has self-signed SSL for internal encryption (defense in depth)
3. **Node.js App** - Runs on HTTP internally (port 5055), no SSL needed
4. **Double SSL** - Common in government environments for extra security

## ⚙️ Nginx SSL Configuration

### File: `/etc/nginx/sites-available/homestay.hp.gov.in`
```nginx
server {
    listen 80;
    server_name homestay.hp.gov.in;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name homestay.hp.gov.in;

    # SSL Configuration
    ssl_certificate /opt/ssl-backup/hptourism.crt;
    ssl_certificate_key /opt/ssl-backup/hptourism.key;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    # Max upload size
    client_max_body_size 50M;

    location / {
        proxy_pass http://localhost:5055;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

### File: `/etc/nginx/sites-available/hptourism` (Default Server)
```nginx
server {
    listen 80 default_server;
    server_name _;
    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl default_server;
    server_name _;

    # SSL Configuration
    ssl_certificate /etc/nginx/ssl/hptourism.crt;
    ssl_certificate_key /etc/nginx/ssl/hptourism.key;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    # Security Headers (CSP)
    add_header Content-Security-Policy "
        default-src 'self';
        script-src 'self' 'unsafe-inline' 'unsafe-eval'
            https://eservices.himachaltourism.gov.in
            https://himaccess.hp.gov.in
            https://sso.hp.gov.in;
        style-src 'self' 'unsafe-inline' https://fonts.googleapis.com;
        font-src 'self' https://fonts.gstatic.com;
        img-src 'self' data: blob:
            https://eservices.himachaltourism.gov.in
            https://himaccess.hp.gov.in
            https://sso.hp.gov.in;
        connect-src 'self'
            https://eservices.himachaltourism.gov.in
            https://himaccess.hp.gov.in
            https://sso.hp.gov.in;
        frame-src 'self'
            https://himaccess.hp.gov.in
            https://sso.hp.gov.in;
        form-action 'self'
            https://himkosh.hp.nic.in
            https://himkosh.hp.gov.in;
        object-src 'none';
    " always;

    server_tokens off;

    location / {
        proxy_pass http://127.0.0.1:5055;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-Proto https;
        
        # Hide app's CSP to use Nginx's permissive CSP
        proxy_hide_header Content-Security-Policy;
    }
}
```

> This allows HimKosh form submissions and SSO iframe functionality.

## 🔄 Certificate Renewal

### Current Certificate Expires: January 8, 2027

Since this is a **self-signed certificate**, renewal requires:

1. **Generate new certificate** (on PROD server):
```bash
openssl req -x509 -nodes -days 365 -newkey rsa:2048 \
    -keyout /opt/ssl-backup/hptourism.key \
    -out /opt/ssl-backup/hptourism.crt \
    -subj "/C=IN/ST=HP/L=Shimla/O=HP_Tourism/OU=IT/CN=homestay.hp.gov.in"
```

2. **Copy to backup location**:
```bash
cp /opt/ssl-backup/hptourism.crt /etc/nginx/ssl/
cp /opt/ssl-backup/hptourism.key /etc/nginx/ssl/
```

3. **Test and reload nginx**:
```bash
nginx -t && systemctl reload nginx
```
