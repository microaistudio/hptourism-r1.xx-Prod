## 1. Registration & Account Management

### User Story 1.1: New User Registration
**As a** homestay owner  
**I want to** create an account on the portal  
**So that** I can apply for homestay registration certificate

#### Test Scenarios

**[TC-001] Valid Registration - All Fields**
- **Pre-condition:** User is on registration page
- **Steps:**
  1. Enter valid full name
  2. Enter valid email (unique)
  3. Enter valid mobile (10 digits, unique)
  4. Set password (min 8 chars, with uppercase, lowercase, number)
  5. Confirm password (matching)
  6. Click Register
- **Expected Result:** Account created, verification email sent, redirect to login page
- **Status:** [ ] Pass [ ] Fail

**[TC-002] Duplicate Email Validation**
- **Pre-condition:** Email already exists in database
- **Steps:**
  1. Attempt registration with existing email
  2. Submit form
- **Expected Result:** Error message "Email already registered"
- **Status:** [ ] Pass [ ] Fail

**[TC-003] Duplicate Mobile Validation**
- **Pre-condition:** Mobile number already exists
- **Steps:**
  1. Attempt registration with existing mobile
  2. Submit form
- **Expected Result:** Error message "Mobile number already registered"
- **Status:** [ ] Pass [ ] Fail

**[TC-004] Password Strength Validation**
- **Test Data:**
  - Weak: "123456"
  - Only lowercase: "abcdefgh"
  - Missing uppercase: "abcdef12"
  - Valid: "Test@1234"
- **Expected Result:** Weak passwords rejected with clear feedback
- **Status:** [ ] Pass [ ] Fail

**[TC-005] Password Mismatch**
- **Steps:**
  1. Enter password: "Test@1234"
  2. Enter confirm password: "Test@123"
  3. Submit
- **Expected Result:** Error "Passwords do not match"
- **Status:** [ ] Pass [ ] Fail

#### Edge Cases

**[TC-006] Network Timeout During Registration**
- **Steps:** Simulate slow network, submit registration
- **Expected Result:** Loading indicator shown, timeout handled gracefully
- **Status:** [ ] Pass [ ] Fail

**[TC-007] Special Characters in Name**
- **Test Data:** "O'Brien", "José García", "राज कुमार"
- **Expected Result:** System accepts valid unicode characters
- **Status:** [ ] Pass [ ] Fail
